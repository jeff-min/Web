manager-windows
==> Bitnami 를 이용해서 설치한 여러 프로그램들을 관리하는 데 고움을 주는 프로그램.
[참고로 manager을 이용하면 웹서버를 켜고 끌 수 있음]

  1. http://127.0.0.1/index.html
    주소를 쳐도 똑같은 화면 나와야함 (확인)

    2. Internet Protocol Address : 컴퓨터 주소
    http : 통신규약, 웹페이지를 웹서버와 웹브라우저가 서로 주고받기 위한 약속
    화면이 나온다면 index.html 이라는 어떠한 파일을 화면에 출력해 준것.

  2. index.html 파일 찾기

    3. Apache 가 설치되어 있는 디렉토리 그안에
    htdocs ('hypertext documents')의 약자 즉 웹 페이지 저장되어진곳(웹 페이지 문서)
  • 웹 서버는 htdocs라는 디렉토리에서 파일을 찾도록 설정이 기본적으로 되어 있는 상태
  • 주소를 입력해서 웹 페이지를 보는 것과 / 파일 열기를 통해서 웹 페이지를 보는 것의 차이는 무엇일까?

    • 즉)
    # http:// 는 통신규약, 웹페이지를 웹서버와 웹브라우저가 서로 주고받기 위해
    # 통신 규약인 HyperText Transfer Protocol 를 이용해서 데이터를 가져오는 것.

    # file:// 는 파일에 있는 걸 직접 여는 것

두 대의 컴퓨터가 서로 웹 페이지를 주고 받는 방법을 살펴보자

일단 보고싶은 파일을 요청해야하니 다른 컴퓨터의 특정 IP Address를 알아야한다.
웹 서버가 설치되어 있는 컴퓨터가 192.168.0.13 라고 한다면
웹 브라우저를 통해서 192.168.0.13 IP로 접속 가능

일단 두 개의 웹 브라우저를 실행함


주소가 127.0.0.1 라고 치는것은 자기 자신을 의미함
모든 컴퓨터가 자기 자신을 127.0.0.1 이라고 약속 되어져 있어서

주소가192.168.0.13(서버가 설치된 컴퓨터 IP Address) 라고 치는것은.
웹 서버있는 파일을 읽어서 전송해줘 웹 브라우저가 열어 보여준 것.
