http://info.cern.ch/

Web Browser 과 Web Sever

가령 Web Sever 가 http://info.cern.ch 란 주소를 갖고 있고 / index.html 이란 파일을 하드디스크에 담고 있다면

Web Browser에서 http://info.cern.ch/index.html으로 Web Sever에게 신호를 보내
(웹 서버가 가지고 있는 주소로 파일을 요청함)
요청한 파일의 코드가 도착하면 코드를 읽어서 웹 브라우저가 읽어서 해석해서
화면에 보여주는게 우리가 알고 있는 웹 사이트이다.
==>이게 웹의 동작원리

위와 같이 Web Browser은 요청(request)을 Web Sever 응답(response)을

즉)
Web Browser => Client / Web Sever => Sever 로 칭함
